Troubleshooting Tips:

  1. For the Restaurant Recommender app, you need a valid Google Maps API key
  2. Replace "YOUR_API_KEY" in index.html with your actual API key
  3. Make sure all files (index.html, app.js, styles.css) are in the same directory
  4. Open index.html directly in your browser instead of using xdg-open
